## 프레임을 넘어, 당신의 세계로, 뮤지엄? 뭐지엄? (Museum? Whatisum?)

박물관 작품을 살아있는 페르소나로 변환하여 관람객과 대화하고 연결합니다.

[Demo](https://www.youtube.com/shorts/INb94UmLhWc)

## 프로젝트 소개

뮤지엄? 뭐지엄?은 AI 기반 페르소나를 통해 예술 작품과 유물을 살아있는 캐릭터로 변환하여 박물관 경험을 혁신합니다. 관람객은 스마트폰 카메라를 전시물에 비추기만 하면, 작품이 자연스럽고 유익한 대화를 나누는 인터랙티브 캐릭터로 변신합니다. RAG 기반 AI와 개인화된 아바타를 활용하여 현재 2.47%에 불과한 박물관의 낮은 디지털 참여율 문제를 해결하고, 특히 Z세대 관람객들에게 몰입감 있고 교육적이며 재미있는 경험을 제공합니다.

## 문제 정의

### 현재 박물관의 문제점

- **낮은 디지털 참여율**: 박물관 앱의 평균 이용률은 2.47%에 불과
- **제한적인 인터랙션**: 기존 오디오 가이드나 텍스트 설명은 일방향적이고 지루함
- **젊은 세대의 박물관 이탈**: 디지털 경험에 익숙한 Z세대는 전통적 박물관 방식에 흥미를 잃음

### 시장 기회

- **글로벌 박물관 기술 시장**: 62억 달러(2024) → 174억 달러(2033)
- **AR/VR이 게임 체인저**: 2024년 박물관 기술 수익의 30% 이상 차지
- **검증된 효과**: 연구에 따르면 모바일 AR은 78.21%의 지식 보존율을 달성하며 사용자 만족도와 플로우 경험을 크게 향상

## 핵심 기능

### 1. 카메라 기반 작품/유물 인식
- 컴퓨터 비전을 활용한 실시간 작품/유물 식별
- 스마트폰 카메라로 전시물을 비추면 즉시 인터랙션 시작

### 2. 아바타 페르소나 + 음성(TTS)
- 각 작품이 고유한 성격을 가진 캐릭터로 변환
- 역사적 맥락을 생생하게 전달하는 자연스러운 음성 대화
- 의인화된 스토리텔링을 통한 감성적 연결

### 3. RAG 기반 대화 엔진
- 큐레이션된 박물관 데이터베이스로부터 정확한 정보 제공
- 사용자 관심사에 맞춰 적응하는 자연스러운 대화 흐름
- 일반 LLM이 어려워하는 마이너한 작품/유물에 대한 전문 지식

### 4. 직관적인 UX 플로우
- 간단한 사용자 여정: 스캔 → 대화 → 채팅 기록 확인 → 다음 작품 스캔
- 복잡한 네비게이션이나 다운로드 불필요
- 박물관 환경에 최적화된 매끄러운 경험

## MVP 범위

### 현재 구현 사항
- **5개 주요 작품**: 일반 LLM이 잘 다루지 못하는 작품에 집중
- **카메라 인식**: 실시간 작품 식별
- **2D 아바타 인터랙션**: 음성 합성이 포함된 시각적 페르소나
- **채팅 히스토리**: 작품과의 과거 대화 내역 검토

### 검증 목표
- 마이너한 작품에 대한 정보 전달 정확도 검증
- 사용자 참여도 및 유지율 테스트
- 지식 보존율 향상 측정
- 페르소나 진정성에 대한 사용자 피드백 수집

## 혁신성 및 차별화

### 우리의 차별점

**전통적 박물관 앱** (2.47% 사용률)
- 정적인 텍스트 및 오디오 가이드
- 일방향 정보 전달

- 양방향 인터랙티브 대화
- 개인화된 스토리텔링과 감성적 연결

## 비즈니스 모델

### B2B2C 접근 방식
- **박물관과 파트너십**: 기관별 라이선스 모델
- **방문객에게 무료**: 진입 비용 장벽 없음
- **수익원**: 
  - 박물관 라이선스 비용
  - 프리미엄 기능 (향후)

### 시장 진출 전략
1. **파일럿 단계**: 2-3개 소규모 박물관과 파트너십
2. **검증**: 지표 및 추천사 수집
3. **확장**: 주요 박물관 및 문화 기관으로 확대
4. **국제화**: 글로벌 시장을 위한 현지화

## 향후 로드맵

### 2단계: 향상된 AR 경험

- **아바타의 제스처**: 화면 속 아바타의 자연스러운 움직임
